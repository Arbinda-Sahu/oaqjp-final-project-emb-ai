# Emotion Detector Web App

## Project Description
This project is an AI-based web application that detects emotions from user input text using Python and Flask.

## Features
- Detects emotions (joy, anger, sadness, fear)
- REST API using Flask
- Unit testing included
- Error handling implemented
- Static code analysis using pylint

## Technologies Used
- Python
- Flask
- IBM Watson NLP (or mock)
- unittest
- pylint

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Test API using Postman or curl
